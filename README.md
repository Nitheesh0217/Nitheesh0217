### 🏛️ [D Scent House — Luxury E-Commerce & Automation Engine](https://github.com/Nitheesh0217/d-scent-house)
> Enterprise-grade, full-stack e-commerce storefront and lifecycle marketing automation engine built for a luxury Arabic fragrance brand in South Florida.

**Stack:** Next.js 14 · TypeScript · Supabase PostgreSQL 15 · Python · Stripe · Resend · Hugging Face · Anthropic Claude · Vercel

**What makes it different:**
- 🔒 **Financial Guardrails** (`guardrails.ts`) — hardcoded margin floors (70%/62%/55%) and category-specific discount caps enforced at the code layer, not just the UI
- ⚙️ **Database-backed Job Queue** — async lifecycle marketing using PostgreSQL `FOR UPDATE SKIP LOCKED` instead of expensive message brokers (RabbitMQ/SQS)
- 🤖 **AI Content Studio** — Founder control center with one-click marketing presets, AI-generated copy (Instagram, TikTok, Email) via Llama 3.1, and auto-publish to storefront
- 🔁 **Zero-Downtime AI Fallback** — Anthropic Claude → Hugging Face fallback pipeline that auto-routes on key failure
- 📬 **5 Automated Lifecycle Flows**: Abandoned Cart Recovery · Post-Purchase Scent Ceremony · Cross-Sell · VIP Vault Recognition · 90-Day Win-Back
