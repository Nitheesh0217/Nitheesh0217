> **The most technically complex project.** Enterprise-grade full-stack e-commerce platform + self-hosted marketing automation engine for a luxury Arabic fragrance brand in South Florida. Built to replace Klaviyo, Shopify, and Hootsuite — zero SaaS fees.

[![Live Storefront](https://img.shields.io/badge/Live%20Storefront-d--scent--house.vercel.app-black?style=flat&logo=vercel)](https://d-scent-house.vercel.app)
[![Admin Studio](https://img.shields.io/badge/Admin%20Studio-Live%20Demo-6366f1?style=flat&logo=vercel)](https://d-scent-house.vercel.app/admin/studio)

**Stack:** `Next.js 14` `TypeScript` `Supabase PostgreSQL 15` `Python` `Stripe` `Resend` `Hugging Face` `Anthropic Claude` `Vercel`

| Engineering Highlight | What It Solves |
|---|---|
| 🔒 **Financial Guardrails** (`guardrails.ts`) | Hardcoded margin floors (70/62/55%) + discount caps — prevents human error at code level, not just UI |
| ⚙️ **PostgreSQL Job Queue** (`FOR UPDATE SKIP LOCKED`) | Replaces RabbitMQ/SQS — async lifecycle marketing with zero duplicate sends across concurrent workers |
| 🤖 **AI Content Studio** | One-click marketing presets, Llama 3.1 copy generation (Instagram/TikTok/Email), auto-publish to storefront |
| 🔁 **Zero-Downtime AI Fallback** | Anthropic Claude → Hugging Face auto-routes on key failure — no manual intervention needed |
| 📬 **5 Lifecycle Marketing Flows** | Abandoned Cart · Post-Purchase · Cross-Sell · VIP Vault ($300 LTV) · 90-Day Win-Back |

- 📊 **“Command Central” Admin CRM** — live lead pipeline with KPI cards (pipeline value, conversion rate), filterable status phases (New → Contacted → Quoted → Scheduled → Completed), and Export Intel (CSV)
- 🤖 **AI Assistant** — service recommendation engine with bundle psychology and anchoring
- 📱 **Mobile Action Bar** — persistent sticky CTA for call / WhatsApp / quote
- 📍 **Service Area coverage** — Miami, Doral, Kendall, Tamiami, Coral Gables, Miami Beach
